This page uses the ReverbRecords API.

It was built with Grape and runs on Rack.

The table below contains the data from a CSV file. When you press one of these buttons, an AJAX call to the Grape API is made. The app then re-reads the data from the file, sorts it as requested, and rebuilds this table. The field there will POST your input string to /records and add either it to the database if it's valid or display an error message if there's a problem with it.
